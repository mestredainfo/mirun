# MIRun

MIRun is software that runs scripts developed in PHP for Linux (Debian and Ubuntu) and Windows desktops.

## System Requirement

- Architecture: x64

### Linux

- Debian 12 or higher
- Ubuntu 22.04 or higher

### Windows

- Windows 10 or higher
- Visual C++ Redistributable 14.42 (Windows)
